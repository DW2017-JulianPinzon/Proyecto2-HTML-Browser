HTML

Historia

HTML (HiperText Markub Language) es un lenguaje de programación para la elaboración de páginas web, que consta de la utilización de corchetes angulares (<>) los cuales se utilizan para traducir la estructura y la información en forma de texto, con la posibilidad de agregar imágenes, videos y otros tipos multimedia. Hoy en día HTML puede utilizar para complementarse hojas de estilo cascada (CSS), java script entre otros.

El origen de HTML se remonta al físico Tim Berners-lee, un trabajador de la organización europea para la investigación nuclear (CERN) que a finales de 1989 propuso el lenguaje HTML junto con el protocolo http, para así crear un medio para poder compartir información entre diversos físicos de la época que trabajaban por todo el mundo. El primer documento que se publicó de manera formal, sobre la estructura HTML se publicó en el año 1991 con el nombre de: “HTML Tags”, este documento aun disponible contiene las etiquetas que se usan en HTML.

Versiones

La primera versión en salir fue el HTML 1, sin embargo, no fue considerado una versión estándar y por eso muchas veces se reconoce al HTML como el primero.

  • HTML 2: Presentado en el año 1995, no soportaba tablas y donde la declaración explicita de los elementos body, html y head es opcional.
  • HTML 3.2: Versión presentada en el año de 1997 gracias al organismo de estandarización llamado W3C (World Wide Web Coonsortium). El cual incorporo los avances de las páginas web como applets de java y texto que fluye alrededor de las imágenes.
  • HTML 4.0: Lanzada en 1998. Entre sus novedades se incluían las hojas de estilos CSS, la posibilidad de incluir pequeños programas o scripts en las páginas web, mejora en la accesibilidad de las paginas diseñadas, tablas complejas y mejoras en los formularios.
  • HTML 4.01: Salió a la luz en el año 1999 como una revisión y actualización de la versión HTML 4.0, no incluye novedades significativas.
  • HTML 5: Quinta revisión importante del lenguaje básico de la World wide web, el cual especifica dos variantes de sintaxis para HTML: el clásico HTML (text/html), y la variante conocida como HTML 5 y una variante XHTML conocida como sintaxis XHTML 5, en la cual se centro W3C para su desarrollo.
